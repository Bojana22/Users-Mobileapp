# Student App – Classroom Presence System
Android апликација за евиденција на присуство на предавања и вежби преку NFC технологија.

Опис
Student App е дел од системот Classroom Presence System. Оваа апликација им овозможува на студентите брзо, едноставно и сигурно да го евидентираат своето присуство на настава преку допирање на нивниот телефон до уредот на професорот (со користење на NFC технологија).

Функционалности
Брза најава: Сигурна најава со студентски профил (проверка на улога – само студенти).
Дигитален студентски индекс/картичка: Телефонот симулира студентска картичка со помош на NFC HCE технологија.
Едноставно евидентирање: Регистрација на присуство со само еден допир до телефонот на професорот додека трае сесијата.
Интуитивен кориснички интерфејс: Преглед на статус за успешна валидација и повратна информација во реално време.
Оптимизирана сигурност: Заштита од злоупотреби и криптиран пренос на податоците при скенирање.

Технологии
Јазик и рамка: Java + XML (Android)
NFC / HCE (Host Card Emulation): Симулација на картичка преку IsoDep / APDU протоколи.
